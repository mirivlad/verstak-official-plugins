## Highlights

- Files and Notes now rename unique targets directly through atomic non-overwriting moves, while real name conflicts remain visible in the form.
- File duplication no longer mistakes every candidate copy name for an existing file and chooses the first available name safely.
- Regression smoke tests cover production-style empty metadata responses, successful renames, real conflicts, and duplication.

## Главное

- Плагины «Файлы» и «Заметки» переименовывают свободные назначения атомарным перемещением без перезаписи, а реальные конфликты имён остаются видимыми в форме.
- Создание копии файла больше не считает каждое возможное имя уже занятым и безопасно выбирает первое свободное.
- Регрессионные smoke-тесты проверяют характерный для production пустой ответ метаданных, успешное переименование, реальные конфликты и создание копии.

## Packages / Пакеты

Install the archive matching the desktop platform into Verstak's plugins directory, replacing the previous official plugins bundle. The same updated plugins are bundled with the desktop patch release.

Установите архив для своей desktop-платформы в каталог плагинов Верстака, заменив предыдущий комплект официальных плагинов. Те же обновлённые плагины включены в patch-релиз desktop-приложения.
