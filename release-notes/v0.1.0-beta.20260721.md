## Highlights

- Sync settings preserve the paired username and clearly indicate when the
  password is stored securely without putting the password back into the form.
- Connection checks reuse the stored pairing token instead of attempting an
  empty-password login.
- The sync status item now reports running, successful, and failed states with
  useful transfer counts and the last successful synchronization time.
- Browser Inbox lists nested semantic Deals only when the Browser plugin is
  active for that Deal, and external links open through the supported desktop
  API.
- Settings panels and filter toolbars follow the shared responsive layout:
  wide content with comfortable padding and whole filter groups wrapping to a
  second row when required.

## Главное

- Настройки синхронизации сохраняют имя подключённого пользователя и явно
  показывают, что пароль хранится безопасно, не возвращая сам пароль в форму.
- Проверка подключения использует сохранённый токен вместо попытки входа с
  пустым паролем.
- Индикатор синхронизации показывает выполнение, успех и ошибку, полезные
  счётчики передачи и время последней успешной синхронизации.
- Browser Inbox показывает вложенные семантические Дела только тогда, когда в
  них активен плагин «Браузер», а внешние ссылки открываются через публичный API
  desktop-приложения.
- Панели настроек и строки фильтров используют общую адаптивную раскладку:
  доступную ширину с нормальными отступами и перенос всей группы фильтров на
  вторую строку при необходимости.

## Packages / Пакеты

Install the archive matching the desktop platform into Verstak's plugins
directory, replacing the previous official plugins bundle.

Установите архив для своей desktop-платформы в каталог плагинов Верстака,
заменив предыдущий комплект официальных плагинов.
