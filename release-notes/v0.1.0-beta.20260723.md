## Highlights

- Added the official `verstak.import` plugin for current DokuWiki and Obsidian content.
- The four-step settings workflow selects a folder or ZIP/TAR backup, analyzes it without writing, proposes an editable structure, and imports only after explicit review.
- DokuWiki pages are converted to Markdown with headings, lists, tables, code/nowiki blocks, formatting, page links, and media links. Current revisions and media are imported; history and service data are omitted.
- Obsidian Markdown, aliases, wiki links, headings, block references, embeds, and ordinary files are preserved or rewritten for the proposed destination. `.obsidian` is omitted and nested archives remain ordinary files.
- Every run is isolated under `Импортировано/<DokuWiki|Obsidian> — timestamp`; naming conflicts receive a numeric suffix.
- Imported pages are never scanned or moved as secrets. The plugin shows one general warning that credentials or other private data may require manual review after import.

## Главное

- Добавлен официальный плагин `verstak.import` для актуального содержимого DokuWiki и Obsidian.
- Четырёхшаговый мастер в настройках выбирает папку или ZIP/TAR-архив, анализирует источник без записи, предлагает редактируемую структуру и импортирует её только после явного подтверждения.
- Страницы DokuWiki переводятся в Markdown с заголовками, списками, таблицами, code/nowiki-блоками, форматированием, ссылками на страницы и media. Импортируются текущие ревизии и media; история и служебные данные исключаются.
- Markdown, aliases, wiki links, headings, block references, embeds и обычные файлы Obsidian сохраняются или переписываются под итоговую структуру. `.obsidian` исключается, вложенные архивы остаются обычными файлами.
- Каждый запуск изолирован в `Импортировано/<DokuWiki|Obsidian> — timestamp`; при совпадении имени добавляется числовой суффикс.
- Страницы не сканируются и не переносятся в Secrets. Плагин показывает одно общее предупреждение, что логины, пароли и другие приватные данные нужно проверить вручную после импорта.

## Known limitation / Известное ограничение

Unsupported DokuWiki plugin syntax is preserved in the Markdown note and reported for manual review instead of being interpreted.

Неподдерживаемый синтаксис плагинов DokuWiki сохраняется в Markdown-заметке и помечается для ручной проверки, а не интерпретируется.

## Packages / Пакеты

Install the archive matching the desktop platform into Verstak's plugins directory, replacing the previous official plugins bundle. The same plugins are bundled with the matching desktop prerelease.

Установите архив для своей desktop-платформы в каталог плагинов Верстака, заменив предыдущий комплект официальных плагинов. Эти же плагины включены в соответствующий prerelease desktop-приложения.
