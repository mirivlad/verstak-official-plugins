## Highlights

- Todo now lists semantic Deals through the public workspace API and excludes ordinary folders and Deals where Todo is disabled.
- Default Editor wraps long lines by default and persists the localized wrap toggle without changing the text written to the vault.
- Files now supports safe copy and move workflows between Deals, including folder trees, visible collision errors, and Deal drop targets backed by an existing `Files` directory.

## Главное

- Todo теперь получает семантические Дела через публичный workspace API и исключает обычные папки и Дела, в которых Todo выключен.
- Default Editor по умолчанию переносит длинные строки и сохраняет локализованный переключатель, не меняя записываемый в vault текст.
- «Файлы» теперь безопасно копируют и перемещают данные между Делами, включая деревья папок, видимые ошибки конфликтов и перенос на Дело с существующей папкой `Files`.

## Packages / Пакеты

Install the archive matching the desktop platform into Verstak's plugins directory, replacing the previous official plugins bundle. The same updated plugins are bundled with the matching desktop patch prerelease.

Установите архив для своей desktop-платформы в каталог плагинов Верстака, заменив предыдущий комплект официальных плагинов. Те же обновлённые плагины включены в соответствующий patch-prerelease desktop-приложения.
